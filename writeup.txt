MICHAEL'S IMPLEMENTATION
------------------------------------

Control Flow Code Generation
- Implemented codegen_conditional() - generates MIPS assembly for
  if/else statements using unique labels
  * Evaluates condition into $t0
  * Uses beq $t0, $zero to branch around then-block
  * Emits j end_label to skip else-block when present
  * Handles both if-only and if/else forms
- Implemented codegen_loop() - generates MIPS assembly for while loops
  * Emits loop_start label before condition evaluation
  * Uses beq $t0, $zero to exit loop when condition is false
  * Jumps back to loop_start after body
- Implemented codegen_for() - generates MIPS assembly for for loops
  by desugaring into init + while loop structure
  * Emits init assignment before loop label
  * Evaluates condition at top of loop
  * Emits body followed by update assignment before jumping back
- Implemented codegen_statement_node() - helper that unwraps STATEMENT
  wrapper nodes so control flow bodies dispatch correctly
- Added COMPOUNDSTMT handling to codegen_statement() so { } blocks
  inside if/else/while/for bodies work correctly

For Loop Language Support
- Added KWD_FOR token to scanner.l
- Added forStmt grammar rule to parser.y parsing
  for(var = expr; expr; var = expr) statement
- Added FORSTMT node kind to tree.h enum
- Wired forStmt into the statement rule in parser.y

Register Clobbering Fix
- Fixed a bug in expression code generation where nested expressions
  would overwrite intermediate values stored in $t1
- Replaced move $t1, $t0 pattern with stack push/pop (addi $sp / sw / lw)
  in codegen_expression(), codegen_addExpr(), and codegen_term()
  so left operands are preserved across recursive right operand evaluation

Testing and Integration

3. COMPILATION INSTRUCTIONS
============================

The provided Makefile is used without modification.

To compile the project:
    make clean && make

To run the compiler on a source file:
    ./obj/mcc <input_file.mC>

To generate assembly for all sample tests:
    ./obj/mcc Sample_Test/test1.mC
    ./obj/mcc Sample_Test/test2.mC
    ./obj/mcc Sample_Test/test3.mC
    ./obj/mcc Sample_Test/test4.mC
    ./obj/mcc Sample_Test/test5.mC
